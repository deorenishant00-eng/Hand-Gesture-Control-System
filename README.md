# ✋ Hand Gesture Control System

A real-time Hand Gesture Control System built using Python, OpenCV, and MediaPipe. This project detects and tracks hand landmarks through a webcam, recognizes predefined gestures, and performs system actions without requiring a physical keyboard or mouse.

# 🚀 Features
Real-time hand detection and tracking
Gesture recognition using MediaPipe Hands
Touchless computer interaction
High-speed webcam processing
Easy to customize with new gestures
User-friendly and lightweight
# 🛠️ Technologies Used
Python
OpenCV
MediaPipe
NumPy
PyAutoGUI (for system control)
CVZone (optional)
# 📌 Applications
Touchless Human-Computer Interaction
Smart Home Control
Presentation Control
Virtual Mouse
Gesture-Based Automation
Accessibility Solutions
# 📂 Project Workflow
Capture live video from webcam.
Detect hand landmarks using MediaPipe.
Identify hand gestures.
Map gestures to specific system commands.
Execute actions in real time.
